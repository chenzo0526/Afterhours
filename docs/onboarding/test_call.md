# Test Call Instructions

Purpose:
Confirm the caller experience and the handoff content before go-live.

Steps:
1) Place a test call through the forwarding path.
2) Confirm the spoken company name is correct.
3) Verify questions are calm, brief, and capture essential details.
4) Confirm the summary format and delivery channel.
5) Confirm emergency disclaimer is delivered when appropriate.

Pass criteria:
- Company name correct
- Summary contains required fields
- Notification delivered to primary on-call

If any item fails, update the configuration and re-run the test call.
