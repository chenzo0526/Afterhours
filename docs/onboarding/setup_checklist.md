# Afterhours Setup Checklist

Required inputs (must be completed):
- Spoken company name (exactly how callers should hear it)
- Main business number
- Primary on-call contact (name, mobile, email)
- Backup on-call contact (name, mobile, email)
- Owner contact for fallback (name, mobile, email)
- After-hours definition + timezone
- Service area boundaries (cities/zip codes covered + exclusions)
- Emergency disclaimer acknowledgment (direct life-safety emergencies to 911)
- Summary delivery preference (SMS, email, or both)
- Call recording preference (on/off + retention expectation)

Forwarding method (choose one):
- Carrier forwarding after hours
- Phone system schedule forwarding
- Dedicated after-hours line
