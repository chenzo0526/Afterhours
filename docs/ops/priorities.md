# Priority Definitions

P0 — Active outage or safety risk
- Calls not answered at all
- Notifications failing for all businesses
- Data loss or privacy impact

P1 — Partial outage or severe degradation
- Calls answered for some businesses but not others
- SMS failures with email still delivering
- Significant delays in summaries

P2 — Non-urgent issue with workaround
- Minor formatting issue in summaries
- Slow notification delivery with eventual success
